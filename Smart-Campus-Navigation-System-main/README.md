## 📖 Overview

Smart Campus Navigation System is an AI-powered Digital Twin platform that helps students, faculty, and visitors navigate university campuses efficiently.

The application combines interactive GIS maps, intelligent routing, AI assistance, and real-time campus information into a modern, responsive interface inspired by Google Maps.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Campus-Navigation-System.git
```

```bash
cd Smart-Campus-Navigation-System
```

## Install Dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file.

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
VITE_MAPBOX_ACCESS_TOKEN=YOUR_MAPBOX_TOKEN
```

---

## Run Locally

```bash
npm run dev
```

---

## Build

```bash
npm run build
```
